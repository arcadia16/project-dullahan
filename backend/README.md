# How to run?

Currently project supports master-server / agent functionality (local only for development)
Modes share same codebase, so just install requirements and launch via shell scripts in separate terminals.

> [!NOTE]
> Project utilizes EXPORT functionality, so do remember to launch in separate terminal instance, but in the same folder.

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r ../requirements.txt
```
